🚀 Pop!_OS

"For developers who think tiling makes them productive."

Who It's For

Target User

Pop!_OS is built for developers, engineers, and power users who want a polished, modern Linux desktop that gets out of their way and lets them code. Made by System76, it's opinionated in the best way.

  • Developers who live in multiple terminal windows
  • Machine learning engineers (GPU support is first-class)
  • Power users who want a tiling window manager without the setup pain

Strengths

What Pop!_OS Does Best

🪟 Auto-Tiling

Pop!_OS has a tiling window manager built right into GNOME. Windows tile automatically. No config files, no i3 setup, no hand-holding needed.

🎮 GPU Support

Pop!_OS ships a separate ISO with Nvidia drivers baked in. Zero driver drama. Your GPU works on day one — for gaming, ML, or creative work.

⚡ Fast & Focused

Pop!_OS strips out the distractions. No Amazon app, no pre-installed games. It's clean, fast, and designed for focus.

Getting Started

Basics

Pop!_Shop — App Store

Pop!_OS has a curated app store called Pop!_Shop. Install software graphically with one click, or use APT in the terminal — Pop!_OS is Ubuntu-based, so it's all compatible.

Installing via APT

Pop!_OS uses the same package manager as Ubuntu. All your Ubuntu knowledge transfers:

sudo apt update
sudo apt install <package-name>

Tiling Keyboard Shortcuts

Once you enable Auto-Tile, windows snap automatically. Use Super + Enter to toggle tiling mode, and arrow keys to navigate windows.

Power User

Advanced Features

COSMIC Desktop (Next-Gen)

System76 is building a brand-new desktop environment called COSMIC, written in Rust. Pop 24.04 onwards will ship it — a fully tiling, keyboard-first desktop built from scratch for speed and extensibility.

Recovery Partition

Pop!_OS ships with a separate recovery partition. If your system breaks, you can reinstall without a USB drive. No re-downloading from scratch.

Flatpak by Default

Pop!_OS supports Flatpak natively, giving you the latest app versions in sandboxed containers independent of the system's base packages.

flatpak install flathub <app-id>
